\# Recipe Book App



\## Project Description

Recipe Book is a simple Android mobile application developed using Java and XML. The app helps users manage recipes through organized screens such as home, recipe details, add recipe, update recipe, login, signup, and profile.



\## Technologies Used

\- Java

\- XML

\- Android Studio

\- Firebase Firestore

\- Git

\- GitHub



\## Main Features

\- Home screen with recipe list

\- Recipe cards

\- Add new recipe

\- Update existing recipe

\- View recipe details

\- User login and signup

\- User profile screen



\## Project Structure

The project is organized into packages:

\- Adapter

\- Model

\- View

1\. Clone the repository.

2\. Open the project using Android Studio.

3\. Sync Gradle files.

4\. Run the app on an emulator or Android device.
